# Tic Tac Toe

A simple and interactive Tic Tac Toe game built with HTML, CSS, and JavaScript. 
This project demonstrates fundamental web development skills, including DOM manipulation, event handling, and responsive design.

---

## Features

- **Interactive Gameplay**: Two-player mode on the same device.
- **Responsive Design**: Works seamlessly on desktops, tablets, and mobile devices.
- **Reset Functionality**: Start a new game at any time.
- **Winning Logic**: Detects winner or draw and highlights the result.

---

## Technologies Used

- **HTML**: For structuring the web page.
- **CSS**: For styling and layout.
- **JavaScript**: For game logic and interactivity.

---

## Getting Started

Follow these steps to get the project up and running on your local machine:

### Prerequisites

- A modern web browser (e.g., Chrome, Firefox, Edge).
- Basic knowledge of Git (optional).

### Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/parths19/TicTacToe.git
   ```

2. Navigate to the project directory:

   ```bash
   cd TicTacToe
   ```

3. Open the `index.html` file in your browser:

   ```
   open index.html
   ```
   Or drag and drop the file into your browser.

---

## Usage

1. Open the game in your browser.
2. Player X starts the game by clicking any available cell.
3. Players take turns to place their marks (X or O).
4. The game announces a winner or declares a draw when applicable.
5. Click the "Reset" button to restart the game.
